Introduction

JavaScript is a versatile and widely-used programming language primarily used to create interactive and dynamic content on web pages. Developed by Brendan Eich in 1995, it has become one of the core technologies of the World Wide Web, alongside HTML and CSS. JavaScript allows developers to implement complex features such as animations, form validation, and asynchronous communication with servers.

Syntax and Basics

// Declare a variable
let message = 'Hello, JavaScript!';

// Function to display a message
function greet(name) {
    return Hello, ${name}!;
}

// Call the function
console.log(greet('World'));

Key Features

<- Dynamically Typed: Variables do not need explicit type definitions.

<- Event-Driven: Supports handling events like clicks, keypresses, etc.

<- Asynchronous Programming:
    Promises, async/await for non-blocking operations.

<- DOM Manipulation:
    Access and modify HTML elements via document object model.

<- Cross-Platform:
    Runs on various environments, including browsers and servers (Node.js).

<- Integrations:
    Works seamlessly with APIs for fetching and sending data.

Q. Animations with setInterval:

let pos = 0;
function moveBox() {
    const box = document.getElementById('box');
    pos += 1;
    box.style.left = pos + 'px';
}
setInterval(moveBox, 10);
