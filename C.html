Introduction

C is a powerful general-purpose programming language that was developed by Dennis Ritchie at Bell Labs in 1972. It is widely used for system programming, developing operating systems, embedded systems, and applications requiring high performance. C provides low-level access to memory and operates close to the hardware, making it highly efficient.

Syntax and Details

                    #include <stdio.h>  // Preprocessor directive to include standard I/O functions
                    
                    int main() {
                        printf("Hello, World!"\n);  // Output statement to print Hello, World!
                        return 0;  // Exit the program
                    }

Key Features

                    <-Case-Sensitive  : Variable names and keywords are case-sensitive.
                    
                    <-Semicolon ;  : Marks the end of a statement.
                    
                    <-Preprocessor Directives  : Start with # and are executed before the 
                    compilation, e.g., #include, #define.
                    
                    <-Functions  : The building blocks of C programs. Execution starts from main().
                    
                    <-Data Types  :
                                 Primitive: int, float, char, double
                                 Derived: Arrays, Pointers, Structures, Unions
                                 Void: No data type, often used in functions with no return value.

                    <-Control Structures:
                                Conditional Statements: if, else, switch
                                Loops: for, while, do-while

                    <-Operators:
                                Arithmetic: +, -, *, /, %
                                Relational: ==, !=, <, >, <=, >=
                                Logical: &&, ||, !

                    <-Uses:-
                    
                    ~System Programming : Operating systems like Linux, Unix.
                    ~Embedded Systems : Microcontroller and microprocessor programming.
                    ~Game Development : Rendering engines and performance-critical components.
                    ~Compilers : C is used to develop compilers for other programming languages.
                    ~Database Systems : Examples include MySQL.
                    
                    <-Common Inside Components:-
                    
                    Standard Libraries:
                    stdio.h: Input/output functions (e.g., printf, scanf)
                    stdlib.h: General utilities (e.g., memory allocation with malloc, free)
                    string.h: String manipulation (e.g., strlen, strcpy)
                    
                    Memory Management:
                    malloc, calloc, realloc, free
                    
                    Pointers:
                    Enable dynamic memory allocation and manipulation of variables directly.
                    Example: int *ptr = &var;
                    
                    Structures and Unions:
                    Structure: Groups different data types together.
                    
                    struct Point {
                        int x;
                        int y;
                    };
                    
                    Union: Memory-efficient grouping of variables.
                    
                    Examples:
                    
                    Q.Hello World Program:
                    
                    #include <stdio.h>
                    
                    int main() {
                        printf("Hello, World!\n");
                        return 0;
                    }

                    Q.Finding the Largest of Three Numbers:
                    
                    #include <stdio.h>
                    
                    int main() {
                        int a, b, c, largest;
                    
                        printf("Enter three numbers: ");
                        scanf("%d %d %d", &a, &b, &c);
                    
                        largest = (a > b && a > c) ? a : (b > c ? b : c);
                    
                        printf("The largest number is %d\n", largest);
                        return 0;
                    }
                    
                    Q.Factorial of a Number (Using Recursion):
                    
                    #include <stdio.h>
                    
                    int factorial(int n) {
                        if (n == 0 || n == 1) return 1;
                        return n * factorial(n - 1);
                    }
                    
                    int main() {
                        int num;
                    
                        printf("Enter a number: ");
                        scanf("%d", &num);
                    
                        printf("Factorial of %d is %d\n", num, factorial(num));
                        return 0;
                    }
                    
                    Q.Swap Two Numbers Using Pointers:
                    
                    #include <stdio.h>
                    
                    void swap(int *a, int *b) {
                        int temp = *a;
                        *a = *b;
                        *b = temp;
                    }
                    
                    int main() {
                        int x = 10, y = 20;
                    
                        printf("Before swapping: x = %d, y = %d\n", x, y);
                        swap(&x, &y);
                        printf("After swapping: x = %d, y = %d\n", x, y);
                    
                        return 0;
                    }
