Introduction

C++ is a general-purpose programming language developed by Bjarne Stroustrup in 1983. It is an extension of the C language, adding Object-Oriented Programming (OOP) capabilities. Known for its high performance, flexibility, and versatility, C++ is widely used in system programming, game development, real-time simulations, and application development.

                    Object-Oriented: Encapsulation, Inheritance, Polymorphism, and Abstraction.
                    <-Standard Template Library (STL) : Predefined templates for data structures and algorithms.
                    <-Strong Typing : Variables must be declared before use.
                    
                    <-Data Types : int, float, double, char, bool, etc.

                    <-Operators : Similar to C (+, -, *, /, %, ++, --).

                    <-Control Structures : if, else, switch, for, while, do-while.

                    <-Object-Oriented Concepts:

                      Class and Object:
                      
                      class Car 
                        {
                          public:
                              string brand;
                              int year;
                        };
                      
                      int main() {
                          Car myCar;
                          myCar.brand = "Toyota";
                          myCar.year = 2020;
                          std::cout << myCar.brand << " " << myCar.year << std::endl;
                          return 0;
                      }
                      
                    <-Namespaces :
                      Prevent name conflicts using namespace:
                      
                      namespace ns1 {
                          int value = 10;
                      }
                      
                      int main() {
                          std::cout << ns1::value << std::endl;
                          return 0;
                      }
                      
                      
                    <-Uses
                      ~System Programming : Writing operating systems, drivers, and hardware interfaces.
                      ~Game Development : Popular in creating high-performance gaming engines.
                      ~GUI Applications : Used for developing desktop applications like Adobe Photoshop.
                      ~Competitive Programming : STL provides fast and efficient solutions.
                      ~Simulations and Modeling : Scientific computing, real-time simulations, and CAD tools.
                      
                    <-Common Inside Components

                      STL (Standard Template Library):
                      
                      Containers: vector, list, map, set
                      Algorithms: sort, reverse, binary_search
                      Iterators: Enable traversal of containers.
                      
                    <-Memory Management:
                      
                      Dynamic memory: new and delete
                      
                      int* ptr = new int(5);
                      delete ptr;
                      Exception Handling:
                      
                      
                      try {
                          throw std::runtime_error("Error occurred");
                      } catch (const std::exception& e) {
                          std::cout << e.what() << std::endl;
                      }
                      
                      Inheritance:

                      class Parent {
                          public:
                              void display() {
                                  std::cout << "Parent class" << std::endl;
                              }
                      };
                      
                      class Child : public Parent {};
                      
                      int main() {
                          Child obj;
                          obj.display();
                          return 0;
                      }

                    <-Sorting an Array Using STL:
                      
                      #include <iostream<
                      #include <algorithm<
                      
                      int main() {
                          int arr[] = {3, 1, 4, 1, 5, 9};
                          int n = sizeof(arr) / sizeof(arr[0]);
                      
                          std::sort(arr, arr + n);
                      
                          for (int i = 0; i < n; i++)
                              std::cout << arr[i] << " ";
                          return 0;
                      }
