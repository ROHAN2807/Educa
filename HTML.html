Introduction

HTML (HyperText Markup Language) is the standard language used to create and design webpages. It was created by Tim Berners-Lee in 1991 and serves as the backbone of the World Wide Web, providing the structure for websites. HTML uses a system of tags to define and organize content, including headings, paragraphs, links, images, and more.

Basic Structure

<!DOCTYPE html>  
<html>           
    <head>       
        <title>Page Title</title>
    </head>
    <body>       
        <h1>Hello World</h1>
        <p>This is a paragraph.</p>
    </body>
</html>

Key Features

- Platform-Independent: Runs on any browser or operating system.
- Tag-Based Language: Uses tags to define structure and style.
- Easy Integration: Can be combined with CSS and JavaScript.
- Accessibility Support: Includes semantic elements for screen readers.
- Multimedia: Supports embedding images, videos, and audio.

Examples of Tags:
    - Structural Tags: <html>, <head>, <body>
    - Text Formatting: <h1>-<h6>, <p>, <b>, <i>
    - Multimedia: <img>, <audio>, <video>
    - Links and Navigation: <a>, <nav>

Commonly Used Elements

- <div> and <span>: Used for grouping and styling content.
- <form>: Enables user input with elements like <input>, <button>, and <textarea>.
- <table>: Creates tables for structured data.
- <ul> and <ol>: Used for unordered and ordered lists, respectively.
- <iframe>: Embeds another HTML page within a page.

Example:
<div>
    <p>This is inside a div.</p>
</div>
