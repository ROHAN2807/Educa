Introduction

Ruby is a high-level, interpreted, general-purpose programming language designed by Yukihiro "Matz" Matsumoto in the mid-1990s. Known for its simplicity and productivity, Ruby emphasizes human-friendly syntax and flexibility. It is dynamically typed and supports multiple programming paradigms, including object-oriented, functional, and procedural styles. Ruby is most famously associated with the Ruby on Rails framework, which revolutionized web development.

Syntax and Details

                   # This is a comment in Ruby

# Define variables

Key Features

                    Object-Oriented: Everything in Ruby is an object.
Dynamic Typing: Variable types are determined at runtime.
Interpreted Language: Executes code line by line without compilation.
Readable Syntax: Designed to be close to natural language.

Object-Oriented Concepts:

Class and Object:

class Car
  attr_accessor :brand, :year

  def initialize(brand, year)
    @brand = brand
    @year = year
  end

  def details
    "#{brand} - #{year}"
  end
end

my_car = Car.new("Toyota", 2020)
puts my_car.details

Factorial Using Recursion:


def factorial(n)
  return 1 if n <= 1
  n * factorial(n - 1)
end

puts factorial(5)
Fibonacci Sequence:


def fibonacci(n)
  a, b = 0, 1
  n.times { a, b = b, a + b }
  a
end

10.times { |i| print "#{fibonacci(i)} " }
