Introduction

Data Structures and Algorithms (DSA) form the foundation of computer science and programming. Data Structures are ways to organize and store data efficiently, while Algorithms are step-by-step procedures to perform tasks or solve problems. DSA is crucial for writing optimized, scalable, and high-performance code. It plays a significant role in problem-solving, competitive programming, and system design.

Syntax and Details

                   Data Structures:

Linear Structures: Data is arranged in a sequential order.
Examples: Arrays, Linked Lists, Stacks, Queues.
Non-Linear Structures: Data is arranged hierarchically or in a non-linear order.
Examples: Trees, Graphs, Heaps.
Algorithms:

Time Complexity: Measures algorithm efficiency based on input size (O(n), O(log n), etc.).
Space Complexity: Measures memory usage of an algorithm.
Types:
Searching: Linear Search, Binary Search.
Sorting: Bubble Sort, Quick Sort, Merge Sort.
Graph Algorithms: BFS, DFS, Dijkstra’s Algorithm.
Common Terminologies:

Big-O Notation: Describes the worst-case performance.
Recursion: A function calling itself.
Dynamic Programming: Solves problems by breaking them into subproblems and reusing results.

Key Features

                    Efficient Data Management: Optimizes data storage and retrieval in software systems.
Problem Solving: Used in competitive programming, coding interviews, and complex problem scenarios.
Database Systems: Implementation of indexing and search trees.
Networking: Routing algorithms like Dijkstra’s and spanning trees.
Real-Time Applications: Priority queues for task scheduling.

def bubble_sort(arr):
    for i in range(len(arr)):
        for j in range(len(arr) - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr

Stack Implementation:

class Stack:
    def _init_(self):
        self.stack = []

    def push(self, item):
        self.stack.append(item)

    def pop(self):
        if not self.is_empty():
            return self.stack.pop()
        return "Stack is empty"

    def is_empty(self):
        return len(self.stack) == 0

s = Stack()
s.push(10)
s.push(20)
print(s.pop())  # Output: 20
Binary Search:

def binary_search(arr, target):
    low, high = 0, len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1

nums = [1, 3, 5, 7, 9]
print(binary_search(nums, 5))  # Output: 2
Breadth-First Search (BFS):

from collections import deque

def bfs(graph, start):
    visited = set()
    queue = deque([start])

    while queue:
        node = queue.popleft()
        if node not in visited:
            visited.add(node)
            print(node, end=" ")
            queue.extend(graph[node])

graph = {

    'D': [],
    'E': [],
    'F': []
}

Merge Sort:

def merge_sort(arr):
    if len(arr) > 1:
        mid = len(arr) // 2
        left = arr[:mid]
        right = arr[mid:]

        merge_sort(left)
        merge_sort(right)

        i = j = k = 0
        while i < len(left) and j < len(right):
            if left[i] < right[j]:
                arr[k] = left[i]
                i += 1
            else:
                arr[k] = right[j]
                j += 1
            k += 1

        while i < len(left):
            arr[k] = left[i]
            i += 1
            k += 1

        while j < len(right):
            arr[k] = right[j]
            j += 1
            k += 1

arr = [12, 11, 13, 5, 6, 7]
merge_sort(arr)
print(arr)  # Output: [5, 6, 7, 11, 12, 13]
Dijkstra’s Algorithm:

import heapq

def dijkstra(graph, start):
    pq = [(0, start)]  # (distance, node)
    distances = {node: float('infinity') for node in graph}
    distances[start] = 0

    while pq:
        current_distance, current_node = heapq.heappop(pq)

        for neighbor, weight in graph[current_node]:
            distance = current_distance + weight
            if distance < distances[neighbor]:
                distances[neighbor] = distance
                heapq.heappush(pq, (distance, neighbor))

    return distances

graph = {
    'A': [('B', 1), ('C', 4)],
    'B': [('A', 1), ('C', 2), ('D', 6)],
    'C': [('A', 4), ('B', 2), ('D', 3)],
    'D': [('B', 6), ('C', 3)]
}
print(dijkstra(graph, 'A'))  # Output: {'A': 0, 'B': 1, 'C': 3, 'D': 6}
