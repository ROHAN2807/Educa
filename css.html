Introduction

CSS (Cascading Style Sheets) is a stylesheet language used for describing the presentation of a document written in HTML or XML. Developed by the W3C in 1996, CSS allows web developers to control the layout, colors, fonts, and overall appearance of web pages.

Syntax and Details

/* Basic Syntax of CSS */
selector {
    property: value;
}

/* Example */
body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
    margin: 0;
}

Key Features

<-Selector Types:->
    - Universal Selector: * { margin: 0; }
    - Type Selector: p { color: blue; }
    - Class Selector: .example { font-size: 20px; }
    - ID Selector: #header { text-align: center; }

<-Box Model:->
    - Margin: Space outside the border.
    - Border: Surrounds the padding and content.
    - Padding: Space between the content and border.
    - Content: The actual content inside the box.

<-Positioning:->
    - Static: Default positioning.
    - Relative: Positioned relative to its normal position.
    - Absolute: Positioned relative to its nearest positioned ancestor.
    - Fixed: Positioned relative to the viewport.

<-Responsive Design:->
    - Media Queries: Adjust styles based on device width and resolution.
    - Flexible Grid: Use CSS Grid or Flexbox for responsive layouts.

<-Common Uses:->
    ~Styling web pages with colors, fonts, and layouts.
    ~Creating responsive designs that adapt to different screen sizes.
    ~Adding animations and transitions for interactivity.
    ~Defining themes for consistent appearance across websites.

<-CSS Frameworks:->
    ~Bootstrap
    ~Tailwind CSS
    ~Foundation

<-Examples:->

Q. Styling Text:

h1 {
    color: #243B55;
    font-size: 36px;
    text-align: center;
}

Q. Creating a Button:

button {
    background-color: #4CAF50;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 5px;
    cursor: pointer;
}

button:hover {
    background-color: #45a049;
}

Q. Responsive Design:

@media (max-width: 768px) {
    body {
        font-size: 14px;
    }
}
